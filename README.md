El presente proyecto implementa un Sistema de Gestión Inmobiliaria desarrollado en Java, enfocado en la correcta aplicación de patrones de diseño GoF, principios SOLID y GRASP.
El sistema permite registrar inmuebles, calcular precios según diferentes estrategias, generar contratos de venta o alquiler y notificar cambios mediante observadores.
Además, integra patrones como Builder, Strategy, Factory Method, Observer, State, Adapter y Facade para garantizar un código modular, mantenible y escalable.
Su arquitectura por capas separa responsabilidades, facilitando futuras extensiones y manteniendo bajo acoplamiento en todo el proyecto.
